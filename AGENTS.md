# AGENTS.md

Instructions for Codex working in this repository.

1. Keep implementation minimal.
2. Prefer native Linux desktop APIs where practical.
3. Prefer `WebKitGTK` for the ChatGPT UI unless clearly unsuitable.
4. Optimize for Lubuntu/LXQt compatibility first.
5. Do not add telemetry or analytics.
6. Do not intercept credentials.
7. Avoid invasive DOM automation unless a milestone explicitly requires a small, auditable amount.
8. Avoid heavy frameworks unless they are clearly justified.
9. Make the smallest reasonable change per step.
10. Stop after each milestone and summarize changes.
